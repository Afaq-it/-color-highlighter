- XML
- HTML
- JavaScript
- TypeScript
- Properties
- YAML
- Python
- Ruby
- Java (ColorUiResource)
- Kotlin (ColorUiResource)
- Scala (ColorUiResource)
- Go
- Objective C
- Swift

More to come!
